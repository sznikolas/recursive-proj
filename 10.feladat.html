Rúdvágás probléma - Rod cutting problem


Feladat leírása: Adott egy n hüvelyk hosszúságú rúd és egy ártömb, amely tartalmazza az n-nél kisebb méretű darabok árait. Határozza meg a rúd feldarabolásával és a darabok eladásával elérhető maximális értéket. Például, ha a rúd hossza 8, és a különböző darabok értékei a következők, akkor a maximálisan elérhető érték 22 (két 2 és 6 hosszúságú darab feldarabolásával)
length| 1   2   3   4  5    6    7   8
---------------------------------------------------------
price|  1   5   8   9  10   17  17  20


Rúdvágás probléma megoldása C nyelvben
                
//C program a rúdvágás problémára

#include <stdio.h>
#include <stdlib.h>
  
int max(int a, int b)
{
  return (a > b) ? a : b;
}
  
//Globális tömb a memoizálás céljából
int t[9][9];
  
/*Egy rekurzív program, amely a memoizálás 
segítségével valósítja meg a rúdvágási 
problémát (Top-Down)*/
int un_kp(int price[], int length[],
                      int Max_len, int n)
{
  
    /*a maximális ár nulla lesz, ha vagy 
      a rúd hossza nulla, vagy az ár nulla*/
    if (n == 0 || Max_len == 0)
    {
        return 0;
    }
  
    /*Ha a rúd hossza kisebb, mint a maximális hossz, 
      a Max_len figyelembe veszi.Most a nyereségtől 
      függően a Max_len vagy elfogadja, vagy elveti.*/
    if (length[n - 1] <= Max_len)
    {
        t[n][Max_len]
            = max(price[n - 1]
                      + un_kp(price, length,
                              Max_len - length[n - 1], n),
                  un_kp(price, length, Max_len, n - 1));
    }
  
    /*Ha a rúd hossza nagyobb, mint a megengedett méret,
      Max_len, akkor nem vesszük figyelembe.*/
    else
    {
        t[n][Max_len]
            = un_kp(price, length,
                              Max_len, n - 1);
    }
  
    /*A Max_len a kapott maximális értéket adja vissza, 
      Max_len, amely a Max_len oszlopban található.*/
    return t[n][Max_len];
}
  
/*program a fenti funkciók tesztelésére*/
int main()
{
    int price[] = { 1, 5, 8, 9, 10, 17, 17, 20 };
    int n = sizeof(price) / sizeof(price[0]);
    int length[n];
    for (int i = 0; i < n; i++)
    {
        length[i] = i + 1;
    }
    int Max_len = n;
  
    //Függvény hívás
    printf("A maximális kapott érték %d \n",
            un_kp(price, length, n, Max_len));
}
