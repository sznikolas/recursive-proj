Reverse a string - Karakterlánc megfordítása


Feladat leírása:Írj egy rekurzív programot egy adott karakterlánc hatékony visszafordítására C nyelven.
Egy adott karakterláncot könnyen visszafordíthatunk a verem adatszerkezet segítségével. Az ötlet az, hogy a karakterlánc minden egyes karakterét egy verembe toljuk, majd elkezdjük feltölteni a bemeneti karakterláncot (a 0. indexről kezdve) a karakterek kiugrasztásával a veremből, amíg az ki nem ürül. Ahogy az előző bejegyzésben láttuk, egy adott karakterláncot könnyen visszafordíthatunk egy verem adatszerkezet segítségével. Mivel a veremről van szó, könnyen átalakíthatjuk a kódot a hívási verem használatára. Statikus változót használni nem ajánlott. Ezt a problémát statikus változó használata nélkül is könnyen megoldhatjuk. A módszer időbonyolultsága O(n), és O(n) implicit helyet igényel a hívási veremhez, ahol n a bemeneti karakterlánc hossza.


Egy karakterlánc megfordítása rekurzívan 
                
//C program egy karakterlánc megfordítására

#include <stdio.h>
#include <string.h>
 
//Függvény két adott karakter felcserélésére
void swap(char *x, char *y)
{
    char temp = *x;
    *x = *y;
    *y = temp;
}
 
//Rekurzív függvény egy adott karakterlánc visszafordítására
void reverse(char str[], int l, int h)
{
    if (l < h)
    {
        swap(&str[l], &str[h]);
        reverse(str, l + 1, h - 1);
    }
}
 
int main()
{
  //A kívánt karakterlánc megadása
    char str[] = "rekurzio";
    reverse(str, 0, strlen(str) - 1);
    printf("A szó reverse: %s", str);
    return 0;
}
