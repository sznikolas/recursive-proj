Reverse a string - Karakterlánc megfordítása


Feladat: Írjon egy rekurzív programot egy adott karakterlánc hatékony visszafordítására C nyelven.



Karakterlánc megfordítása

Karakterlánc megfordítása - Reverse a string

  Egy karakterlánc megfordítására többféle lehetőségünk is van.

  Egy adott karakterláncot könnyen visszafordíthatunk a verem adatszerkezet segítségével. Az ötlet az, hogy a karakterlánc minden egyes karakterét egy verembe toljuk, majd elkezdjük feltölteni a bemeneti karakterláncot (a 0. indexről kezdve) a karakterek kiugrasztásával a veremből, amíg az ki nem ürül.

  Megoldható beépített metódusok használatával is, ami a C nyelven a strrev() függvény segítségével lehetséges.

  Megoldható az általunk szemléltetett rekurzív módon is, ahol a reverse függvény annyiszor hívja önmagát, amíg az alapfeltételt(base case) el nem érjük. Itt a string 0. elemétől és az utolsó elemétől kezdjük a betűk felcserélését és haladunk a string közepe felé, amíg elérjük a megadott base case feltételt, ekkorra a karakterláncunk megfordítása sikeresen teljesül.


Egy karakterlánc megfordítása rekurzívan 
                
//C program egy karakterlánc megfordítására
#include <stdio.h>
#include <string.h>
 
//Függvény két adott karakter felcserélésére
void swap(char *x, char *y)
{
    char temp = *x;
    *x = *y;
    *y = temp;
}
 
//Rekurzív függvény egy adott karakterlánc visszafordítására
void reverse(char str[], int l, int h)
{
//base case
    if (l < h)
    {
        swap(&str[l], &str[h]);
        reverse(str, l + 1, h - 1);
    }
}
 
int main()
{
  //A kívánt karakterlánc megadása
    char str[] = "rekurzio";
    reverse(str, 0, strlen(str) - 1);
    printf("A szó reverse: %s", str);
    return 0;
}


Forrás
  • Hívjuk meg a rekurzív reverse függvényt a megadott karakterláncunkra.
  • Megállási feltétel: l < h, ami ha (l < h), akkor cseréljük fel az l indexnél lévő karaktert és a h indexnél lévő karaktert, majd növeljük az l indexét egyel és csökkentsük a h indexét 1-gyel. Ha elértük a megállási feltételt a karakterláncunkat sikeresen megfordítottuk.


Az algoritmus leírása

Függvények leírása:

swap → Ez egy egyszerű függvény, aminek segítségével feltudunk cserélni két karaktert.

reverse → Ez a rekurzív függvényünk, ami addig hívja önmagát, míg el nem érjük a megadott base case feltételt. Ezt úgy érjük el, hogy a swap függvényt alkalmazzuk a karakterláncunk első és utolsó karakterére, haladva a karakterláncunk közepe felé, míg elérjük az alapfeltételt.
