Feladat: Írjon programot C nyelvben egy szám faktoriálisának rekurzióval történő kiszámítására.



Faktoriális

Faktoriális

  A matematikában egy n nemnegatív egész szám faktoriálisának az n-nél kisebb vagy egyenlő pozitív egész számok szorzatát nevezzük. Jelölése: n!, amit „n faktoriális”-nak vagy „n faktor”-nak olvasunk ki.

  Az alább látható megoldásnál azt szemléltettük, hogy a rekurzív függvényhívások hogyan tárolódnak a veremben (stack), így könnyebben megérthetjük a rekurzív függvényhívások működését és tárigényét.


Faktoriális számítás C nyelvben rekurzióval

//C program a faktoriális kiszámítására

#include <stdio.h>
//factorial függvény az n faktoriális kiszámítására
int factorial(int n) {
  //alapeset
  if(n <= 1) {
      return 1;
  } else {
    //a factorial függvény rekurzív hívása önmagának
      return n * factorial(n-1);
  }
}

int main() {
  //a kívánt szám megadása
  int n = 7;
  printf("A %d szám faktoriálisa: %d\n" , n , factorial(n));
}


Forrás(1) 

Forrás(2)
  • Hívjuk meg a rekurzív faktorial függvényt egy n egész számra.
  • 1.) Vizsgáljuk meg a base case-t (n <= 1). Ha igen, adjunk vissza 1-et.
  • 2.) Ha nem, akkor hívjuk meg a rekurzív factorial függvényt az (n-1) számra (amíg elérjük a base case-t) , szorozzuk meg az eredményt n-el, és térjünk vissza az értékkel.


Az algoritmus leírása

Függvény leírása:

factorial(int n) → Kezdetben a 7-es értéket adjuk meg argumentumként. Ezután 6 kerül átadásra a factorial(int n) függvénynek ugyanabból a függvényből (rekurzív hívás). Minden egyes rekurzív hívásnál az n argumentum értéke 1-gyel csökken. Ha az n értéke eléri az 1-et (elérjük a base case-t), akkor nincs több rekurzív hívás, visszatérünk a rekurzív függvényhívásokból és megkapjuk az n! eredményét.

Megjegyzés:
0! = 1
1! = 1
