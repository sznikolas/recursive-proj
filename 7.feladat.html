Gyorsrendezés - Quick sort

  A Quicksort egy olyan rendezési algoritmus, amely az oszd meg és uralkodj megközelítésen alapul(hasonlóan a merge sorthoz), ahol egy tömböt altömbökre osztunk egy vezérelem(pivot) kiválasztásával, majd ezeket a részeket az algoritmus rekurzív hívásával rendezi.

  A quicksortnak számos különböző változata létezik, amelyek különböző módon választják ki a pivotot:
    - mindig az első elemet választjuk pivotnak
    - mindig az utolsó elemet választjuk pivotnak
    - egy véletlenszerű elemet választunk pivotnak
    - a mediánt választjuk pivotnak.

  A quicksort kulcsfolyamata a partiton függvény. A particionálás célja, hogy egy adott tömb pivotját a megfelelő helyre helyezzük a tömbbön belül úgy, hogy a pivotnál kisebb elemek a pivot bal oldalára, míg a nagyobbak a jobb oldalára kerülnek.

  Az alábbiakban egy olyan gyorsrendező algoritmust szemléltetünk, ahol a pivot mindig a tömb utolsó eleme lesz.


Gyorsrendezés megoldása C nyelvben 
                
//C program a gyorsrendezés megoldására

#include <stdio.h>

//függvény az elemek cseréjéhez
void swap(int *a, int *b) {
  int t = *a;
  *a = *b;
  *b = t;
}

// függvény a partíció pozíciójának megtalálásához
int partition(int array[], int low, int high) {
  
  //válassza ki a jobb szélső elemet pivotként
  int pivot = array[high];
  
  //mutató a nagyobb elemre
  int i = (low - 1);

  /*a tömb minden egyes elemén végigmegyünk, 
    összehasonlítjuk őket a pivottal*/
  for (int j = low; j < high; j++) {
    if (array[j] <= pivot) {
        
      /*ha a pivotnál kisebb elemet találunk, 
      cseréljük ki az i által mutatott nagyobb elemmel*/
      i++;
      
      //cserélje ki az i elemet a j elemmel
      swap(&array[i], &array[j]);
    }
  }

  //cserélje ki a pivot elemet az i-nél található nagyobb elemmel
  swap(&array[i + 1], &array[high]);
  
  //visszaadja a partíciós pontot
  return (i + 1);
}

void quickSort(int array[], int low, int high) {
  if (low < high) {
    
    //keressük meg azt a pivot elemet, amely
    //a pivotnál kisebb elemek és a pivottól balra helyezkednek el
    //a pivotnál nagyobb elemek és a pivottól jobbra helyezkednek el
    int pi = partition(array, low, high);
    
    //rekurzív hívás a pivot bal oldalán
    quickSort(array, low, pi - 1);
    
    //rekurzív hívás a pivot jobb oldalán
    quickSort(array, pi + 1, high);
  }
}

//függvény a tömb elemeinek kiíratására
void printArray(int array[], int size) {
  for (int i = 0; i < size; ++i) {
    printf("%d  ", array[i]);
  }
  printf("\n");
}

//main függvény
int main() {
  int data[] = {8, 7, 2, 1, 0, 9, 6};
  
  int n = sizeof(data) / sizeof(data[0]);
  
  printf("Rendezetlen tömb\n");
  printArray(data, n);
  
  //quicksort elvégzése az adatokon
  quickSort(data, 0, n - 1);
  
  printf("Rendezett tömb növekvő sorrendben: \n");
  printArray(data, n);
}

 
Forrás
  • A Quicksort egy olyan rendezési algoritmus, amely az oszd meg és uralkodj megközelítésen alapul, ahol
  • Egy tömböt altömbökre osztunk egy pivot elem (a tömbből kiválasztott elem) kiválasztásával.
  • A tömb felosztása során a pivot elemet úgy kell elhelyezni, hogy a pivotnál kisebb elemek a bal oldalon, a pivotnál nagyobb elemek pedig a pivot jobb oldalán maradjanak.
  • A bal és jobb oldali altömböket is ugyanezzel a módszerrel osztjuk fel.
  • Ez a folyamat addig folytatódik, amíg minden altábla egyetlen elemet nem tartalmaz. Ekkor az elemek már rendezettek. Végül az elemek egyesítésével egy rendezett tömb jön létre.
