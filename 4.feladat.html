Hanoi tornyai - Tower of Hanoi


Feladat leírása: A hanoi torony egy matematikai rejtvény. Háromrúdból és különböző méretű korongokból áll, amelyek bármelyik rúdra csúszhatnak. A rejtvény úgy kezdődik, hogy a korongok méretük szerint növekvő sorrendben, rendezett halomban vannak az egyik rúdon, a legkisebb a tetején. Ugyanezt a halmazt kell a harmadik rúdra is felrakni.


Hanoi tornyok megoldása C nyelvben
                
//C program a Hanoi tornyok megoldására

#include <stdio.h>

void towerOfHanoi(int, char, char, char);

int main()
{
    //Lemezek száma
    int num;

    printf("Adja meg a lemezek számát: ");
    scanf("%d", &num);
    printf("A Hanoi-toronyban szereplő lépések sorrendje a következő: \n");
    //A, B, C az oszlopok nevei
    towerOfHanoi(num, 'A', 'C', 'B');
    return 0;
}
void towerOfHanoi(int num, char frompeg, char topeg, char auxpeg)
{
    if (num == 1)
    {
        printf("\n 1. lemez áthelyezése %c oszlopról %c oszlopra", frompeg, topeg);
        return;
    }
    towerOfHanoi(num - 1, frompeg, auxpeg, topeg);
    printf("\n %d. lemez áthelyezése %c oszlopról %c oszlopra", num, frompeg, topeg);
    towerOfHanoi(num - 1, auxpeg, topeg, frompeg);
}


Forrás
  • tower(disk, source, inter, dest)
  • IF disk is equal 1, THEN
    move disk from source to destination
  • ELSE
    tower(disk - 1, source, destination, intermediate) // Step 1
    move disk from source to destination // Step 2
    tower(disk - 1, intermediate, source, destination) // Step 3
  • END


A kód magyarázata

A futtatásakor történő események szemléltetésének egy másik módja a hívásfa. Ez az összes hívás grafikus ábrázolása. Az alább látható ábra egy bináris fagráfot tartalmaz, amit a korongok áthelyezésének könnyebb megértésére hoztunk létre. A START-ból indulva mélységi preorder bejárással járjuk be a gráfot egészen a STOP-ig, ahol a korongok áthelyezése már sikerült.
A hívásfában minden egyes függvényhívást csomópontnak nevezünk. A közvetlenül bármely n csomópont alatt összekapcsolt csomópontok az n függvényhívás által végrehajtott függvényhívásokat jelentik. A teteje alatt például 4(A,B) és 4(B,C) található, mivel ez az a két függvényhívás, amelyet a towerOfHanoi 5(A,C) végrehajt. Alul sok olyan csomópont található, amelyek alatt nincs egyetlen csomópont sem összekapcsolva - ezek az alapeseteket képviselik.
Jelölés: szám(betű,betű)
A szám a korong számát jelöli, míg az első betű, hogy honnan, a második pedig, hogy hova helyezzük a lemezt.
