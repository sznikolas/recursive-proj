Beszúrásos rendezés - Insertion sort


Feladat leírása: Legegyszerűbb formája az egyenes beszúrásos rendezési algoritmus egy tömb elemeinek sorba rendezésére. Az algoritmus k-adik lépése előtt az első k-1 elem már rendezett; a lépés során a k-adik elemet beszúrjuk az első k-1 elem közé az őt nagyság szerint megillető helyre, és a nála nagyobbakat eggyel eltoljuk.


Beszúrásos rendezés megoldása C nyelvben 
                
//C program a beszúrásos rendezésre

#include <stdio.h>

//Rekurzív függvény a beszúrásos rendezéshez
void recursiveInsertionSort(int arr[], int n){
    if (n <= 1)
      return;
    recursiveInsertionSort( arr, n-1 );
    int nth = arr[n-1];
    int j = n-2;
    while (j >= 0 && arr[j] > nth){
      arr[j+1] = arr[j];
      j--;
    }
    arr[j+1] = nth;
}
//main függvény
int main(){
    //a rendezetlen tömbb megadása
    int array[] = {34, 7, 12, 90, 51};
    int n = sizeof(array)/sizeof(array[0]);
    printf("Rendezetlen tömbb:\t");
      for (int i=0; i < n; i++)
    printf("%d ",array[i]);

      recursiveInsertionSort(array, n);
    printf("\nRendezett tömb:\t");
    for (int i=0; i < n; i++)
      printf("%d ",array[i]);
    return 0;
}

A kód magyarázata

Rekurzívan hívjuk meg a recursiveInsertionSort() függvényt az n-1 elemű tömb rendezéséhez az aktuális iterációhoz képest. Ezután ebbe a rendezett tömbbe, amelyet a függvény visszaad, beillesztjük az n-edik elemet a rendezett tömbben elfoglalt helyére.
