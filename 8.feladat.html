Buborékrendezés - Bubble sort


Feladat leírása: A buborékrendezés (angolul: Bubble sort) egy naiv algoritmus, amellyel egy véges (nem feltétlenül numerikus) sorozat – vagy számítástechnikai szóhasználattal élve egy tömb – elemei sorba rendezhetők [(n-1)n]/2 összehasonlítás elvégzésével, ahol n a sorozat elemeinek számát jelenti.
A buborékrendező algoritmus rekurzívan is megvalósítható. Ez látható az alábbi feladatban.


Buborékrendezés megoldása C nyelvben
                
//C program a buborékrendezés megoldására

#include <stdio.h>

//Segédfüggvény értékek cseréjére egy tömb két indexében
void swap(int arr[], int i, int j)
{
    int temp = arr[i];
    arr[i] = arr[j];
    arr[j] = temp;
}
  
//Funkció az `arr` tömb `n` elemének kiírására
void printArray(int arr[], int n)
{
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
        
    }
}
  
//Rekurzív függvény az `arr[i...n]` résztömbbön buborékrendezéshez
void bubbleSort(int arr[], int n)
{
    for (int i = 0; i < n - 1; i++)
    {
        if (arr[i] > arr[i + 1]) {
            swap(arr, i, i + 1);
        }
        //animalashoz segitseg
        //printf("%d \n", arr[i] );
    }
  
    if (n - 1 > 1) {
        bubbleSort(arr, n - 1);
    }
}
  
int main(void)
{
    //Rendezetlen tömbb megadása
    int arr[] = { 10, 5, 8, 4, 1, 9, 3 };
    printf("A rendezett tömbb: ",arr);
    int n = sizeof(arr) / sizeof(arr[0]);
  
    bubbleSort(arr, n);
    printArray(arr, n);
  
    return 0;
}


Forrás
  • Amíg n-1 > 1 rekurzívan hívjuk a buborékrendezést az arr[i...n] résztömbbön n-1-ig.
