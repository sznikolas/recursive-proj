Buborékrendezés

Buborékrendezés - Bubble sort

  A buborékrendezés egy naiv algoritmus(könnyű megérteni és implementálni de nagy az erőforrásigénye - tár és időköltség), amellyel egy tömb elemei sorba rendezhetőek [(n-1)n]/2 összehasonlítás elvégzésével, ahol n a sorozat elemeinek számát jelenti.

  Lényege, hogy két szomszédos elem összehasonlításánál mindig a nagyobb értékű kerüljön hátulra, ha növekvő sorrendbe rendezünk. Az algoritmus újra és újra végigjárja a tömböt, először a végéig, majd miután a legnagyobb elem a végére került, már csak az utolsó előtti elemig és így tovább. A legnagyobb értékű elemek mindig a tömb végére kerülnek.

  A buborékrendező algoritmus rekurzívan is megvalósítható, de általánosságban ez nem egy rekurzív rendezési algoritmus, mint a merge sort és a quick sort. Mi egy rekurzív megoldását mutatjuk be, hogy bizonyítsuk, hogy az általában nem rekurzióval megoldott algoritmusok megoldhatóak rekurzívan is. Ez látható az alábbiakban.


Buborékrendezés megoldása C nyelvben
                
//C program a buborékrendezés megoldására

#include <stdio.h>

//Segédfüggvény értékek cseréjére egy tömb két indexében
void swap(int arr[], int i, int j)
{
    int temp = arr[i];
    arr[i] = arr[j];
    arr[j] = temp;
}
  
//Funkció az arr tömb n elemének kiírására
void printArray(int arr[], int n)
{
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
        
    }
}
  
//Rekurzív függvény az arr[i...n] résztömbbön buborékrendezéshez
void bubbleSort(int arr[], int n)
{
    for (int i = 0; i < n - 1; i++)
    {
        if (arr[i] > arr[i + 1]) {
            swap(arr, i, i + 1);
        }
        //animáláshoz segítség
        //printf("%d \n", arr[i] );
    }
  
    if (n - 1 > 1) {
        bubbleSort(arr, n - 1);
    }
}
  
int main(void)
{
    //Rendezetlen tömbb megadása
    int arr[] = { 10, 5, 8, 4, 1, 9, 3 };
    printf("A rendezett tömbb: ",arr);
    int n = sizeof(arr) / sizeof(arr[0]);
  
    bubbleSort(arr, n);
    printArray(arr, n);
  
    return 0;
}


Forrás
  • Amíg n-1 > 1 rekurzívan hívjuk a buborékrendezést az arr[i...n] résztömbbön n-1-ig.


Az algoritmus leírása

swap(int arr[], int i, int j) → Segédfüggvény az értékek felcserélésére egy tömb két indexében.

printArray(int arr[], int n) → Függvény a rendezett tömb kiíratására.

bubbleSort(int arr[], int n) → asd

Megjegyzés:
int n = sizeof(arr) / sizeof(arr[0]); -> megkapni a tömb hosszát
